Μερικές επισημάνσεις
--------------------

* Παρακαλούμε να αναρτάτε κείμενο γραμμένο στα *Ελληνικά* και όχι Greeklish.
* Όταν αναρτάτε ένα άρθρο, παρακαλούμε όπως επιλέξετε την σωστή κατηγορία ανάρτησης
* Στην ανάρτησή σας, δώστε μια συνοπτική περιγραφή
* Αν διαφωνείτε με κάποια ανάρτηση ή σχόλιο, ενημερώστε τους διαχειριστές: [+Simos Xenitellis] [3] και [+Salih Emin] [4]
* Σεβαστείτε τους κανόνες ["Ανοιχτού Σεβασμού"] [5]
